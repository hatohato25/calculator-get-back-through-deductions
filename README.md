# 控除還付金計算サイト

サラリーマン向けの所得税控除・還付金を簡単に計算できるWebアプリケーションです。

## 概要

年末調整や確定申告で適用できる各種控除を入力すると、還付される税金の概算額を自動計算します。ブラウザ上で動作するため、個人情報は外部に送信されず、安心してご利用いただけます。

## 主な機能

### 対応している控除項目

- **給与所得**
- **iDeCo（個人型確定拠出年金）**
- **生命保険料控除**（新制度・旧制度対応）
- **地震保険料控除**
- **医療費控除**
- **住宅ローン控除**（居住開始年・住宅種類別対応）
- **特定支出控除**
- **寄附金控除（ふるさと納税等）**

### 便利な機能

- **自動保存**：入力内容はブラウザに自動保存され、再訪問時も引き続き利用できます
- **リアルタイム計算**：入力と同時に還付金額が自動計算されます
- **詳細な計算内訳**：控除額や税額の計算過程を確認できます
- **ヘルプ表示**：各項目にヘルプアイコンがあり、入力方法を確認できます

## 使い方

### 1. 給与所得の入力

年間の給与収入額を入力します。源泉徴収票の「支払金額」欄に記載されている金額です。

### 2. 各種控除の入力

適用できる控除項目を入力します：

- **iDeCo**：年間掛金額を入力
- **生命保険料**：新制度・旧制度を選択し、各保険料を入力
- **地震保険料**：年間支払額を入力
- **医療費**：年間の医療費総額と保険金等の補填額を入力
- **住宅ローン**：年末残高・居住開始年・住宅種類を選択
- **特定支出**：給与所得者の特定支出を入力
- **寄附金**：ふるさと納税等の寄附額を入力

### 3. 還付金の確認

入力した内容に基づいて、以下の情報が表示されます：

- **還付金額**：年末調整または確定申告で戻ってくる税金の概算額
- **所得税額**：控除適用後の所得税額
- **控除額の内訳**：各種控除がどれだけ適用されたか

## 注意事項

### 計算の正確性について

このツールは概算計算を行うものです。以下の点にご注意ください：

- 実際の還付額は、扶養控除や配偶者控除など、このツールで対応していない控除項目によって変動する場合があります
- 税制は年度によって変更される可能性があります
- 正確な税額は、税務署または税理士にご確認ください

### データの取り扱い

- 入力データはすべてブラウザ内（localStorage）に保存されます
- サーバーへの送信は一切行われません
- データを削除したい場合は、ヘッダーの「データクリア」ボタンをクリックしてください

### 対象年度

- 2022年以降の税制に基づいて計算しています
- 住宅ローン控除は居住開始年により控除率・期間が異なります

## 技術情報

### 使用技術

- **フロントエンド**: Solid.js + TypeScript
- **スタイリング**: Tailwind CSS
- **ビルドツール**: Vite
- **テスト**: Vitest
- **データ管理**: localStorage

### ローカル環境での実行

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# テスト実行
npm test
```
